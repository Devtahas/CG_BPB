# NetTools Pro – نسخه Cloudflare
**[English](README_EN.md)** | **فارسی**


... (بقیه محتوای فارسی)

**NetTools Pro** یک نرم‌افزار کامل ویندوزی برای مهندسان شبکه، علاقه‌مندان به حریم خصوصی و هر کسی است که نیاز به دور زدن فیلترینگ یا بهینه‌سازی اتصال خود دارد.  
این برنامه یک کلاینت VPN مدرن، اسکنرهای پیشرفته، تکنیک‌های عبور از DPI، مدیریت DNS و بسیاری ابزارهای دیگر را در یک رابط کاربری ساده و کاربرپسند ترکیب می‌کند.

> **نسخه ۹.۸** – به‌روزرسانی بزرگ از نسخهٔ قدیمی ترمینالی. اکنون یک برنامهٔ دسکتاپ کامل ساخته‌شده با CustomTkinter.

---

## 🚀 ویژگی‌های کلیدی

- **کلاینت VPN** – وارد کردن و مدیریت کانفیگ‌های VLESS, VMess, Shadowsocks, Trojan, Hysteria2, TUIC, WireGuard, SOCKS و HTTP Proxy. پشتیبانی از حالت TUN، Kill Switch، زنجیره Pre‑VPN، Traffic Mimicry و به‌روزرسانی خودکار Xray‑core.
- **اسکنر CF** – سریع‌ترین IPهای کلودفلر را برای Worker تنظیمی پیدا می‌کند. چندرشته‌ای با مدیریت CIDR، DNS و پورت‌های سفارشی. تنظیم خودکار Fragment بر اساس ISP شما.
- **پروکسی تلگرام** – دریافت پروکسی‌های MTProto از کانال‌های تلگرام، پینگ زنده و کپی لینک اتصال با یک کلیک.
- **ابزارها و ژنراتورها** – مجموعه‌ای غنی از ابزارها:
  - تولید UUID / رمز عبور
  - استخراج‌کنندهٔ کانفیگ (پارس کردن JSON وی‌لس)
  - اسکنر حرفه‌ای پورت (TCP connect, SYN stealth, UDP) با تشخیص دستگاه/سیستم‌عامل
  - Profile Maker (ضبط و بازپخش ترافیک وب‌سایت)
  - Datacenter Scanner (پیدا کردن IPهای تمیز ایرانی برای زنجیرهٔ Pre‑VPN)
  - DNS Scanner (تست سرعت و سلامت سرورهای DNS)
  - Fastly Scanner (کشف IPهای لبهٔ Fastly)
  - **پیش‌پردازشگر ترافیک محلی** – تغییر شکل ترافیک برای شبیه‌سازی یک وب‌سایت معمولی قبل از خروج از دستگاه.
- **تست سرعت** – اندازه‌گیری پینگ، دانلود و آپلود با استفاده از زیرساخت Cloudflare.
- **فضای ذخیره‌سازی و دارایی‌ها** – مدیریت متمرکز رنج‌های IP و لیست‌های DNS. مرور و ویرایش تمام فایل‌های کانفیگ تولیدشده.
- **تغییر DNS** – تغییر DNS سیستم با یک کلیک. پشتیبانی از IPv4, DoH, DoT و یک سرور DNS محلی کلودفلر.
- **WARP (AmneziaWG)** – اتصال از طریق Cloudflare WARP با استفاده از درایور AmneziaWG. دارای اسکنر پیشرفته endpoint.
- **Psiphon** – اجرای مخفی Psiphon 3 (بدون پنجرهٔ مرئی).
- **شبکهٔ Tor** – اجرای Tor Expert Bundle مستقیماً از برنامه و مسیردهی تمام ترافیک سیستم.
- **ضد فیلتر (حالت اضطراری)** – زنجیرهٔ بقای چندلایه: شکار DNS ← Tor ← Psiphon ← WireGuard ← هدایت خودکار به اسکنر.
- **حالت گیمینگ** – شتاب‌دهندهٔ سیستم (بستن برنامه‌های پس‌زمینه)، تثبیت‌کنندهٔ پینگ (FEC)، بهینه‌ساز NAT و DNS مخصوص بازی.
- **پیام‌رسان امن** – چت رمزنگاری سرتاسری با TLS 1.3 و کشف دستگاه‌ها در شبکه محلی (LAN).
- **مرورگر امن** – اجرای Chrome/Edge با User‑Agent سفارشی، پشتیبانی از پروکسی، حالت Tor و پرچم‌های ضد اثرانگشت.
- **داشبورد** – شاخص کیفیت شبکه زنده، نمودار سرعت لحظه‌ای، وضعیت سرویس‌ها و نمایش مسیر بسته‌ها.
- **تنظیمات** – به‌روزرسانی هسته‌ها (Xray, GoodbyeDPI)، فعال‌سازی رمزنگاری AES‑256‑GCM برای ذخیره‌سازی، قفل برنامه و تغییر مسیر ذخیره‌سازی.
- **سیستم پلاگین** – گسترش برنامه با نصب پلاگین از فایل ZIP یا فروشگاه آنلاین.

## 📥 نصب

### روش ۱ – فایل EXE آماده (توصیه‌شده)
1. به [صفحهٔ انتشارات](https://github.com/Devtahas/CG_BPB/releases) بروید و آخرین `NetTools_Pro_Release.zip` را دانلود کنید.
2. فایل فشرده را در هر پوشه‌ای استخراج کنید.
3. (اختیاری) فایل‌های هستهٔ مورد نیاز (`xray.exe`, `tor.exe`, `amneziawg.exe`, `psiphon3.exe`) را در پوشهٔ `cores` قرار دهید اگر همراه برنامه نبودند.
4. `NetTools_Pro.exe` را **به‌عنوان Administrator اجرا کنید** (راست‌کلیک ← Run as administrator).  
   *دسترسی ادمین برای تغییر DNS سیستم، نصب TUN و اجرای Kill Switch الزامی است.*

### روش ۲ – اجرا از سورس
1. مخزن را کلون کنید:
   ```bash
   git clone https://github.com/Devtahas/CG_BPB.git
   cd CG_BPB
   ```
2. پایتون ۳.۱۱+ را نصب کنید.
3. وابستگی‌ها را نصب کنید:
   ```bash
   pip install -r requirements.txt
   ```
4. فایل‌های هسته (`xray.exe`, `tor.exe`, `amneziawg.exe`, `psiphon3.exe`) را در پوشهٔ `cores/` قرار دهید (یا بعداً از طریق تنظیمات برنامه دانلود کنید).
5. برنامه را اجرا کنید:
   ```bash
   python main.pyw
   ```

## 📖 راهنمای استفاده

### 🛡️ کلاینت VPN
قلب برنامه.  
- **وارد کردن کانفیگ‌ها:** یک لینک VLESS/VMess و غیره، URL اشتراک یا اسکن QR کد را paste کنید.
- **مدیریت کانفیگ‌ها:** مشاهده، ویرایش، حذف یا "بازسازی" (Revive) کانفیگ‌ها (تحمیل SNI جدید با پروفایل Mimicry).
- **اتصال:** یک کانفیگ انتخاب کنید، در صورت نیاز حالت TUN (VPN سراسری) را فعال کنید و روی **CONNECT** کلیک کنید.
  - **زنجیره Pre‑VPN:** از یک IP تمیز Datacenter به‌عنوان پروکسی میانی استفاده کنید.
  - **Traffic Mimicry:** ترافیک VPN را شبیه یک وب‌سایت معمولی (آپارات، یوتیوب و …) کنید با انتخاب یک پروفایل.
- **تنظیمات پیشرفته DPI:** قبل از اتصال، Fragment, SNI Spoofing, REALITY, FakeTLS و … را فعال کنید.

### ⚡ اسکنر CF
سریع‌ترین IPهای کلودفلر را برای Worker خود بیابید.  
- UUID، میزبان Worker و مسیر WS/gRPC را وارد کنید.
- اسکن را شروع کنید. اسکنر هزاران IP را تست کرده، تأخیر و سرعت را اندازه‌گیری کرده و کانفیگ‌ها و لینک اشتراک آماده تولید می‌کند.
- رنج‌های CIDR، پورت‌های سفارشی و سرورهای DNS را از طریق مدیران داخلی مدیریت کنید.

### ✈️ پروکسی تلگرام
پروکسی‌های MTProto را از یک کانال تلگرام (پیش‌فرض: `ProxyMTProto`) واکشی کنید.  
- روی **Fetch & Ping** کلیک کنید – برنامه کانال را scrape می‌کند، هر پروکسی را پینگ کرده و سریع‌ترین‌ها را نشان می‌دهد.  
- لینک `tg://` مستقیماً در کلیپ‌بورد کپی می‌شود.

### 🛠️ ابزارها و ژنراتورها
- **ژنراتورها:** UUID تصادفی و رمز عبور امن تولید کنید.  
- **Config Extractor:** یک JSON وی‌لس را وارد کنید و بلافاصله UUID، میزبان و مسیر را استخراج کنید.  
- **Port Scanner:** اسکنر حرفه‌ای با سه حالت: TCP Connect، SYN Stealth (نیازمند Scapy) و UDP. پورت‌های باز را شناسایی کرده، سرویس و سیستم‌عامل را حدس زده و دستگاه‌ها (دوربین، روتر و …) را تشخیص می‌دهد.
- **Profile Maker:** ترافیک یک وب‌سایت را ضبط کرده و بعداً برای تولید نویز پس‌زمینه بازپخش کنید.
- **Datacenter Scanner:** IPهای دیتاسنترهای ایرانی را برای یافتن IPهای تمیز که به Worker شما می‌رسند، اسکن می‌کند. کانفیگ‌های Pre‑VPN برای زنجیره‌سازی تولید می‌کند.
- **DNS Scanner:** تأخیر و سلامت چندین سرور DNS را همزمان تست کنید.
- **Fastly Scanner:** مشابه Datacenter Scanner اما IPهای لبهٔ Fastly را هدف قرار می‌دهد.
- **Pre‑Processor:** یک پروکسی SOCKS5 محلی که الگوی ترافیک شما را (اثر انگشت TLS، padding، jitter) تغییر می‌دهد تا شبیه یک سایت لیست‌سفید شود. ایده‌آل برای عبور از DPI.

### 🚀 تست سرعت
اندازه‌گیری زندهٔ پینگ، سرعت دانلود و آپلود با استفاده از شبکهٔ جهانی Cloudflare.

### 💾 فضای ذخیره‌سازی و دارایی‌ها
- **مسیر ذخیره‌سازی:** می‌توانید تمام داده‌ها (کانفیگ‌ها، اشتراک‌ها، تنظیمات) را به یک مسیر دلخواه منتقل کنید.
- **دارایی‌های IP و DNS:** مدیریت متمرکز رنج‌های IP (کلودفلر، دیتاسنتر، Fastly) و سرورهای DNS که توسط همهٔ اسکنرها استفاده می‌شوند.

### 🌍 تغییر DNS
- از لیست بزرگ داخلی ارائه‌دهندگان DNS (Google, Cloudflare, Quad9, Shecan, Electro و …) انتخاب کنید.
- سرورهای IPv4، DoH یا DoT سفارشی اضافه کنید.
- **سرور DNS محلی کلودفلر** را فعال کنید که دامنه‌های کلودفلر را با بهترین IPهای لیست اسکن‌شده پاسخ می‌دهد.
- تست نشت DNS، بررسی DNSSEC، استخراج CNAME و پیکربندی Split DNS.

### 🌪️ WARP (AmneziaWG)
- یک شناسهٔ WARP بسازید و با درایور AmneziaWG (فورک anti‑DPI وایرگارد) متصل شوید.
- از **اسکنر پیشرفته** برای یافتن سریع‌ترین endpoint استفاده کنید.

### 🅿️ Psiphon
کلاینت Psiphon 3 را در پس‌زمینه اجرا کنید. برنامه به‌طور خودکار پنجرهٔ آن را مخفی کرده و اتصال را نظارت می‌کند.

### 🧅 Tor
Tor Expert Bundle را راه‌اندازی کرده، روند bootstrap را مشاهده کرده و کل ترافیک سیستم خود را از طریق شبکهٔ Tor هدایت کنید.

### 🆘 ضد فیلتر (حالت اضطراری)
وقتی همه چیز شکست می‌خورد، این مکانیزم بقا چندین پروتکل را زنجیره می‌کند:
۱. سریع‌ترین DNS را پیدا می‌کند
۲. تلاش برای اتصال با Tor
۳. سقوط به Psiphon
۴. تلاش با WireGuard (AmneziaWG)
۵. در صورت شکست همه، به‌طور خودکار **اسکنر CF** را برای یافتن endpointهای جدید باز می‌کند.

### 🎮 حالت گیمینگ
- **شتاب‌دهندهٔ سیستم:** برنامه‌های پس‌زمینه را می‌بندد و فرآیندهای بازی را در اولویت بالا قرار می‌دهد.
- **تثبیت‌کنندهٔ پینگ:** با استفاده از الگوریتم FEC پکت‌لاس را کاهش می‌دهد.
- **بهینه‌ساز NAT:** تنظیمات TCP ویندوز را برای تأخیر کمتر تنظیم می‌کند.
- **DNS مخصوص بازی:** سرورهای DNS بهینه‌شده برای بازی را اعمال کنید.

### 💬 پیام‌رسان امن
- **میزبانی اتاق:** یک سرور چت رمزنگاری‌شده روی شبکهٔ محلی خود راه‌اندازی کنید. به‌صورت اختیاری رمز اتاق تعیین کنید.
- **عضویت در اتاق:** با وارد کردن IP، پورت و نام کاربری به یک اتاق میزبانی‌شده متصل شوید.
- تمام پیام‌ها **رمزنگاری سرتاسری** (تبادل کلید RSA‑2048 + رمزنگاری AES‑256‑GCM) شده و از طریق TLS 1.3 منتقل می‌شوند.

### 🌐 مرورگر امن
Chrome یا Edge را با تنظیمات حریم خصوصی پیشرفته اجرا کنید:
- User‑Agent سفارشی (تقلید Chrome، Firefox، Safari و …)
- پرچم‌های ضد اثرانگشت (غیرفعال کردن WebRTC، Canvas و …)
- پروکسی داخلی و حالت Tor (از طریق 127.0.0.1:9050)
- مسدودسازی تبلیغات از طریق فایل hosts سیستم

### 📊 داشبورد
یک پنل نظارت زنده که نشان می‌دهد:
- **شاخص کیفیت شبکه** (بر اساس پینگ و پکت‌لاس)
- نمودار سرعت آپلود/دانلود لحظه‌ای
- وضعیت سرویس‌های فعال (VPN، DNS، Tor و …)
- IP عمومی و اطلاعات ISP
- نمایش متحرک مسیر بسته‌ها

### ⚙️ تنظیمات
- **به‌روزرسانی هسته‌ها:** آخرین نسخهٔ Xray‑core و GoodbyeDPI را بررسی و نصب کنید.
- **امنیت:** **رمزنگاری ذخیره‌سازی AES‑256‑GCM** را برای محافظت از تمام کانفیگ‌ها و تنظیمات خود با یک رمز اصلی فعال کنید. می‌توانید **قفل برنامه** را نیز تنظیم کنید.
- **ذخیره‌سازی:** مسیر پیش‌فرض داده‌ها را تغییر دهید (به‌طور پیش‌فرض برای حمل‌پذیری در کنار فایل اجرایی است).
- **ظاهر:** بین تم Dark و Light جابه‌جا شوید؛ زبان را تغییر دهید (انگلیسی / فارسی).

### 🧩 پلاگین‌ها
قابلیت‌های NetTools Pro را با نصب پلاگین‌های شخص ثالث گسترش دهید.  
می‌توانید یک پلاگین را از **فایل ZIP** وارد کنید یا مستقیماً از **فروشگاه آنلاین پلاگین** (نیازمند Cloudflare Worker) دانلود کنید.  
پلاگین‌ها می‌توانند تب‌های جدیدی به بخش‌های موجود اضافه کنند یا در پس‌زمینه اجرا شوند.

## ⚙️ پیکربندی و ذخیره‌سازی

به‌طور پیش‌فرض، تمام داده‌های کاربر در پوشهٔ `NetTools_Data` در کنار فایل اجرایی ذخیره می‌شود.  
می‌توانید این مسیر را از **تنظیمات ← ذخیره‌سازی** تغییر دهید.

فایل‌ها و پوشه‌های مهم:
- `Configs/` – تمام فایل‌های کانفیگ VPN تولید و واردشده (JSON).
- `Subscriptions/` – لینک‌های اشتراک (کدگذاری Base64).
- `Settings/` – تنظیمات برنامه، لیست‌های DNS، رنج‌های IP، پروفایل‌های mimicry و پایگاه دادهٔ رمزنگاری‌شده.
- `cores/` – فایل‌های Xray‑core (`xray.exe`), Tor (`tor.exe`), AmneziaWG (`amneziawg.exe`) و Psiphon (`psiphon3.exe`) را اینجا قرار دهید.

## 🔧 ساخت از سورس

اگر می‌خواهید برنامه را خودتان بسته‌بندی کنید:

۱. **وابستگی‌ها را نصب کنید** (به requirements.txt مراجعه کنید).
۲. **اطمینان حاصل کنید** که پوشه‌های زیر در ریشهٔ پروژه وجود دارند:
   - `cores/` (شامل `xray.exe`, `tor.exe`, `amneziawg.exe`, `psiphon3.exe`)
   - (اختیاری) `xray/` (اگر می‌خواهید Xray را جداگانه باندل کنید)
۳. PyInstaller را اجرا کنید:
   ```bash
   pyinstaller --onefile --noconsole --uac-admin --icon=icon.ico `
       --add-data "cores;cores" `
       --collect-all customtkinter `
       --collect-all pyzbar `
       ...
       --name NetTools_Pro main.pyw
   ```
۴. فایل اجرایی خروجی در پوشهٔ `dist/` قرار خواهد گرفت.
